> Le projet consiste à créer un site front / back imitant le fonctionnement de dealabs.

### 🏠 [Homepage](https://github.com/sainteTrinity/PHP_Dealabs_LAPENDERY_VANDAMME)

## Prerequisites

- PHP 8.1
- Symfony 6.2
- Docker
- Docker-compose
- Composer

## Install

```sh
composer install
```

## Usage

```sh
docker-compose up -d
```
